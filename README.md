# SpeakGenie – Interactive Learning Web Application

## 1. Project Title

**SpeakGenie**

---

## 2. Project Description

SpeakGenie is a frontend web application built using **React and Vite** that provides an interactive learning experience through audio lessons, practice sections, and quiz-based challenges. The project is designed with a modular component structure to ensure scalability, reusability, and maintainability.

This application demonstrates modern React development practices such as component-based architecture, client-side routing, and clean folder organization.

---

## 3. Problem Statement

Many learners struggle with engaging and interactive learning platforms. SpeakGenie aims to solve this problem by offering:

- Audio-based learning
- Practice sections
- Quiz challenges to test understanding

The goal is to make learning more interactive and user-friendly.

## Brief Description of Folder Structure

The project follows a clean and modular folder structure for better readability and maintainability.

- **src/** – Contains the complete source code of the React application.
- **assets/** – Stores static files such as images and SVG icons.
- **Components/** – Contains all reusable components, organized by feature:
  - **Home/** – Homepage components like banner, audio player, and practice section.
  - **Quizpage/** – Components related to quizzes and challenge levels.
  - **Detailspage/** – Component for displaying detailed content.
  - **Layout/** – Shared layout components used across pages.
  - **Router/** – Centralized routing configuration.
- **App.jsx** – Root component of the application.
- **main.jsx** – Entry point where the React app is rendered.
- **App.css & index.css** – Global styling files.
- **public/** – Contains public assets and the main HTML file.

This structure ensures separation of concerns and supports scalable development.
